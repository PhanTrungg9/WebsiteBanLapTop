# WebsiteBanLapTop

## 📌 Giới thiệu

Dự án website bán laptop được phát triển trong quá trình học tập/làm việc trước đây.

## 👨‍💻 Vai trò của tôi
* Chức vụ : trưởng nhóm (leader)
* Tham gia phát triển: fullstack
* Các chức năng đã làm:
  
  * Xây dựng giao diện trang sản phẩm , giao diện quản lý
  * Xử lý API lấy dữ liệu
  * Thiết kế database 

## 📅 Thời gian thực hiện

* 10/2025 - 12/2025

## 🔁 Ghi chú

Đây là phiên bản tôi re-upload lại từ dự án đã thực hiện trước đó để phục vụ mục đích học tập và portfolio cá nhân.

## 🚀 Công nghệ sử dụng

* HTML, CSS, JS / Java / C# / SQL.
